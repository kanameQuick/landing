WonderStars and 2D Image Roleplay

In August 2021, 'Team Teriffic' was founded by myself and began developing 2D Image Roleplay on that same date. I made the game because Custom Image Roleplay (a game that was popular at the time) does not allow anyone to use off-sale images, even if they actually own them. This raises serious privacy concerns among users alike, and I want to make a better alternative for that game, so I did so using all my skills and resources that I have.

2D Image Roleplay only had 2 features at the time: Votekick and the piece de r√©sistance, the Morph. A year later, once 2DIRP became noticed by a handful of people, it was updated to Version 2 with new features and possibilities, including chat customisation and props.

Along with the release of Version 2, Amburn, a proprietary anti-cheat engine used in 2D Image Roleplay, was created to deter and prevent various exploiting attempts from happening in the game. It has been maintained at this point to detect new methods of exploiting, and it has been used in another game called "Comedy World RP Fixed" by Baconify.

Later at this point, 'Team Teriffic' is now called what we know today as 'WonderStars', and later on, 2D Image Roleplay is now at its latest major release: Version 3, with more props, features, and the ability to use a 'Back Image'.

After these years, 2D Image Roleplay has gained a lot of visits from players in certain fandoms, enjoying their lives roleplaying as their favourite characters, and it has become one of the fun and safer alternatives to Custom Image Roleplay that is available in Roblox.
